# StemForge AI - استودیوی جداسازی اجزای آهنگ

## هدف پروژه
StemForge AI یک وب‌سایت استاتیک حرفه‌ای با HTML، CSS، JavaScript، jQuery و Bootstrap است که برای آپلود آهنگ، تحلیل صوتی، نمایش Visualizer و پیش‌نمایش اجزای مختلف آهنگ مثل وکال، بیس، درام و سازها طراحی شده است.

> نکته مهم: جداسازی واقعی و حرفه‌ای Stemها مثل Vocal/Bass/Drums/Instrumental معمولاً نیازمند مدل‌های AI مانند Demucs یا Spleeter و پردازش سنگین سمت سرور/مدل WebAssembly است. این نسخه یک فرانت‌اند حرفه‌ای و آماده اتصال به API است و در مرورگر از فیلترهای فرکانسی Web Audio API برای پیش‌نمایش و خروجی تقریبی استفاده می‌کند.

## ویژگی‌های تکمیل‌شده
- طراحی کاملاً ریسپانسیو و مدرن با فضای دارک/نئونی
- استفاده از Bootstrap RTL، jQuery، Font Awesome و Google Font Vazirmatn
- آپلود فایل صوتی با کلیک یا Drag & Drop
- نمایش اطلاعات فایل: نام، حجم، فرمت و مدت زمان
- پلیر صوتی با کنترل‌های سفارشی Play/Pause، جلو/عقب و Volume
- Visualizer زنده با Canvas و Web Audio API
- حالت‌های Stem Preview:
  - Original Mix
  - Vocal Focus
  - Bass Line
  - Drums / Percussion
  - Instrumental Focus
- مسیریابی گراف صوتی با BiquadFilter در سمت مرورگر
- شبیه‌سازی وضعیت پردازش و Progress Meter
- خروجی WAV سمت کلاینت برای حالت انتخاب‌شده
- بخش توضیح اتصال API برای جداسازی AI واقعی
- پیام‌های Toast برای راهنمایی کاربر

## مدل داده و سرویس ذخیره‌سازی
- هیچ دیتابیس یا سرویس ذخیره‌سازی استفاده نشده است.
- فایل صوتی فقط در مرورگر کاربر خوانده می‌شود.
- برای پخش، از `URL.createObjectURL(file)` استفاده می‌شود.
- برای رندر خروجی، از `OfflineAudioContext` و تبدیل Client-side به WAV استفاده شده است.

## Public URLs
- Production URL: پس از انتشار از طریق Publish tab در دسترس خواهد بود.
- API Endpoints: در این نسخه API واقعی متصل نشده است.

## ویژگی‌های هنوز پیاده‌سازی‌نشده
- جداسازی واقعی AI وکال/بیس/درام/سازها
- اتصال به API واقعی CORS-enabled برای پردازش Stemها
- صف پردازش فایل‌های بزرگ
- نمایش Waveform دقیق زمانی با Peak Data
- کنترل Mix/Mute جداگانه برای هر Stem واقعی
- خروجی ZIP شامل همه Stemهای واقعی

## محدودیت فنی نسخه استاتیک
وب‌سایت استاتیک نمی‌تواند پردازش سنگین سروری یا اجرای مدل‌های بزرگ AI را به‌صورت قابل اعتماد انجام دهد. فیلترهای موجود فقط محدوده‌های فرکانسی را تقویت/کاهش می‌دهند و خروجی آن‌ها جایگزین جداسازی حرفه‌ای AI نیست.

## پیشنهاد مراحل بعدی توسعه
1. انتخاب موتور جداسازی واقعی مثل Demucs، Spleeter، Open-Unmix یا یک API آماده.
2. راه‌اندازی بک‌اند جداگانه برای دریافت فایل صوتی و اجرای مدل AI.
3. ارائه API بدون مشکل CORS برای اتصال فرانت‌اند.
4. بازگرداندن لینک Stemهای واقعی به فرانت‌اند.
5. افزودن قابلیت دانلود همه Stemها و کنترل Mixer چندکاناله.

## نحوه اجرا
کافی است `index.html` را در مرورگر باز کنید. برای بهترین نتیجه، پروژه را از طریق یک سرور استاتیک یا Publish tab اجرا کنید.

## انتشار
برای Deploy و آنلاین کردن سایت، از **Publish tab** استفاده کنید تا پروژه با یک کلیک منتشر شود و لینک نهایی دریافت کنید.
